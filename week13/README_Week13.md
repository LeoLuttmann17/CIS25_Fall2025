 Week 13 - Product Class

 What this program does
This program creates a Product object and prints its details. It shows how classes work in C++.

 Files
- Week13.cpp - main file
- Product.h - header file
- Product.cpp - class functions

 How to run
g++ -o Week13 Week13.cpp Product.cpp
./Week13

 Output
Howdy! Constructor called

Printing details below:
ID: 4567, Name: Ice Cream, Price: $2.99

Howdy! Destructor called

 What I learned
- How to make a class with private variables
- How to use constructors and destructors
- How to separate code into .h and .cpp files
